3.Генератор паролей [5 баллов]  Реализуйте программу рандомной генерации паролей. Программа на вход должна принимать два аргумента через stdin:

  • длина пароля – положительное целое число
  • набор символов, из которых случайным образом генерируется пароль

Sample Input 1:

5 abdefg123456_

Sample Output 1:

1_d3a

Sample Input 2:

10

Sample Output 2:

PasswordGenerator <Length> <Alphabet>

Sample Input 3:

-5 abdefg123456_

Sample Output 3:

PasswordGenerator <Length> <Alphabet>

Sample Input 4:

0 abdefg123456_

Sample Output 4:

PasswordGenerator <Length> <Alphabet>


Memory limit: 256 MB
Time limit: 8 seconds
