Лабораторная работа №2

Целью данной лабораторной работы является первоначальная настройка рабочего окружения, окружения тестирования и изучение основ языка Java.

 

Установка Open JDK для Windows

  1. Скачайте Open JDK 11 для Windows по ссылке http://jdk.java.net/java-se-ri/11
  2. Извлеките zip файл в папку, например, C:\Program Files\Java\ , при этом полный путь до jdk будет C:\Program Files\Java\jdk-11
  3. Установите путь до выполняемых файлов в системную переменную PATH, например "C:\WINDOWS\system32;C:\WINDOWS;C:\Program Files\Java\jdk-11\bin"
  4.  Установите переменную JAVA_HOME - C:\Program Files\Java\jdk-11
  5. Для проверки правильности установки выполните в терминале команду java -version

 

Настройка IDE

  1. Скачайте и установите IntelliJ IDEA Community Edition https://www.jetbrains.com/idea/download/#section=windows
  2. Запустите IDE, шаги по кастомизации можно пропустить 
  3. Сконфигурирейте SDK по умолчанию: Welcome screen -> Configure -> Structure for New Projects.  Выберите Add New SDK и укажите путь до JDK.  Project SDK и Project language level должы буть установлены как скриншоте. Главное чтобы везде была 11 версия. 

  4. Проверьте настройку IDE, создав и выполнив консольное приложение. Welcome Screen -> Create New Project
