Настройка окружения для тестирования

Можно решать задачу и проверять решение прямо на Stepik или использовать EduTools Plugin, который позволяет проверять решение из IDE.

 

Тестирование с использование Stepik UI

 

Тестирование с использование плагина

1. Установить плагин IDE -> File -> Settings -> Plugins -> EduTool -> Install -> Restart IDE

2. Присоедениться к stepik-курсу в IntelliJ IDEA.  Welcome Screen -> Learn -> Browse Courses или File Menu -> Learn -> Browse Courses, затем выбрать из списка  "Разработка на платформе Java" . Дождитесь загрузки курса. При первом запуске потребуется залогиниться на Stepik. Повтороно залогинться можно с помощью иконки Stepik в правом нижнем углу в IDE

3. После загрузки курса слева в IDE появится план курса с заданиями. При выборе задачи справа появиться ее описание. Внизу также будет кнопка "Check" , после нажатия на которую решение отправляется на сервер Stepik'а для проверки. Результат с деталями можно увидеть под кнопкой.

 

4. После успешной попытки можно увидеть что результат отражен на сайте Stepik.

5. Во время решения заданий крайне рекомендуется делать автоматическую инспекцию вашего кода и исправлять найденные проблемы. Сделать это можно следующим образом Контекстное меню (правой кнопкой по файлу) -> Analyze -> Inspect Code или через пункт меню  Analyze -> Inspect Code. После появится панель Inspection Result c найденными проблема по категориям.
