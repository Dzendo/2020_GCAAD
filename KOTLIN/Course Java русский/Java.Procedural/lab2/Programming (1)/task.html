2.Шифр Цезаря [10 баллов]  Реализуйте шифр Цезаря (сдвиг вправо по алфавиту на key букв) Программа на вход через stdin должна принимать три аргумента:

  • режим работы: -e – кодировнаие,  -d – декодирование
  • ключ шифрования/дешифрования – целое число
  • строка для шифрования/дешифрования

Sample Input 1:

-e 1 dwwdfndwrqfh

Sample Output 1:

exxegoexsrgi

Sample Input 2:

-d 1 exxegoexsrgi

Sample Output 2:

dwwdfndwrqfh

Sample Input 3:

-x 1 dwwdfndwrqfh

Sample Output 3:

CaesarCipher <Mode> <Key> <Text>

Sample Input 4:

-d 1

Sample Output 4:

CaesarCipher <Mode> <Key> <Text>

Sample Input 5:

-e 108 abzyABZY

Sample Output 5:

efdcEFDC


Memory limit: 256 MB
Time limit: 8 seconds
